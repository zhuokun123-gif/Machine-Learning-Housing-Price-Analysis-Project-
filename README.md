# 🏠 California Housing Price Prediction (CSMAI Coursework)

## 📌 Project Overview

The objective is to perform **data analysis, preprocessing, and machine learning modeling** on the California Housing dataset to predict house prices.

## 📂 Dataset

* Dataset: `housing.csv`
* Source: California Housing Dataset
* Contains features such as:

  * Longitude & Latitude
  * Housing Median Age
  * Total Rooms / Bedrooms
  * Population
  * Median Income
  * Ocean Proximity (categorical)

---

## 🔍 Project Workflow

### 1. Exploratory Data Analysis (EDA)

* Dataset inspection (`info()`, `describe()`)
* Analysis of numerical & categorical features
* Distribution visualisation (histograms)
* Correlation analysis

### 2. Data Preprocessing

* Handling missing values
* Feature engineering
* Encoding categorical variables
* Feature scaling

### 3. Model Building

* Train/test split
* Model selection (e.g., Linear Regression, etc.)
* Model training and evaluation

### 4. Evaluation

* Performance metrics (e.g., RMSE)
* Comparison of model performance

## 📊 Key Insights

* Strong correlation between **median income and house price**

* Categorical variable `ocean_proximity` influences pricing
